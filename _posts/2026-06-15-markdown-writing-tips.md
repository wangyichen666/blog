# Markdown 写作小技巧

Markdown 是写博客最顺手的格式。这里整理几个常用技巧，帮助排版更清晰。

## 1. 善用标题层级

合理使用 `#` 到 `######` 构建文章结构，方便读者快速浏览，也利于生成目录。

## 2. 代码块标注语言

为代码块标注语言可以获得语法高亮：

```js
function greet(name) {
  console.log(`Hello, ${name}!`);
}
```

## 3. 表格对齐

用冒号控制列对齐方式：

| 功能      | 语法            | 说明         |
| :-------- | :-------------: | -----------: |
| 左对齐    | `:---`          | 默认         |
| 居中      | `:---:`         | 标题常用     |
| 右对齐    | `---:`          | 数字列适用   |

## 4. 引用与提示

> 引用块适合放置重点提示或他人观点。

## 小结

掌握这些基础技巧，就能写出排版清爽的博客文章了。祝写作愉快！
